## 三条变现路线对比

| | 接单服务 | 内容自媒体 | 领域深耕（球探+卡） |
|---|---|---|---|
| 启动速度 | 快（1-2周） | 慢（1-3月） | 已启动 |
| 收入天花板 | 中（时薪制） | 高（流量放大） | 高（专业壁垒） |
| 技术门槛 | 低 | 极低 | 有（球探经验） |
| 是否需要持续投入 | 是 | 是 | 是 |
| 当前适合你的程度 | 一般 | 可以试 | **最合适** |

## 三个知识点（遇到时再学，不单独安排）

1. **API 是什么**：当你想让网页自动获取实时数据时学（第 2-3 周可能遇到）
2. **Node.js**：当你需要后端处理数据时学（不急着装）
3. **微信小程序**：当你想做微信生态产品时学（第 4 周可选）
