# 球星卡价格预测框架 v2

> 创建日期：2026-07-13
> 用途：每次分析一张球星卡时，按此框架搜集信息并输出预测
> 工作流：用户提卡 → Claude自动搜索填入 → 用户补充 → 输出预测

---

## 工作流程

1. 用户说"帮我分析 XX 球员的 XX 卡"
2. Claude 自动搜索以下信息并填入框架：
   - 球员数据（Transfermarkt、SofaScore、WhoScored）
   - 近期表现和新闻
   - 卡片市场行情（eBay成交价、卡淘、PWCC等）
   - 同类球员卡价对比
3. 标注哪些是「已搜集」、哪些是「需你补充」
4. 用户审核补充后，输出完整预测报告

---

## 信息搜集框架（8大模块）

### 模块一：球员基本面
- 姓名、年龄、国籍
- 位置、惯用脚
- 身高/体型
- 性格/场外形象（星味、媒体曝光、争议风险）
- 踢球风格（观赏型 vs 功能型）

### 模块二：球员表现
- 本赛季数据（出场、进球、助攻、场均评分）
- 近期状态（近5-10场趋势：上升/平稳/下滑）
- 国家队情况（是否主力、出场、大赛表现）
- 俱乐部地位（绝对主力/轮换/替补）
- 职业生涯高光（名场面、决赛表现等）

### 模块三：球员轨迹判断
- 年龄阶段（上升期/巅峰期/下滑期）
- 发展曲线（持续进步/平台期/走下坡）
- 俱乐部环境（教练信任、战术适配、球队稳定）
- 转会可能性（传闻、潜在下家）
- 伤病史（大伤记录、出勤率）

### 模块四：横向对比
- 同位置同龄段全球排名
- 模板球员（发展轨迹像谁）
- 对比球员A/B（同类型、同级别）
- 对比球员的卡价（作为价格锚点）

### 模块五：卡片属性
- 品牌/系列/年份
- 卡片类型（Base/Parallel/Insert/RC/Auto/Patch）
- 编号情况（/xx、是否同背号）
- 评分情况（PSA/BGS/SGC + 分数 + POP存世量）
- 是否为RC（新秀年）
- 卡片品相（角、面、居中、边缘）
- 购入成本

### 模块六：市场行情
- 当前市场价（最近成交价范围）
- 近3个月价格走势（涨/跌/横盘 + 幅度）
- 同球员其他卡表现
- 同类球员卡价锚点
- 在售数量/流动性

### 模块七：催化剂
- 近期大赛（世界杯/欧洲杯/欧冠/美洲杯）
- 转会窗口（转会豪门 = 关注度暴增）
- 个人荣誉（金球/金童/联赛最佳）
- 破纪录可能
- 媒体叙事（话题人物、故事性）

### 模块八：风险因素
- 伤病风险
- 竞争风险（同位置抢主力）
- 俱乐部风险（财政、降级、换帅）
- 市场泡沫风险（是否已被过度炒作）
- 流动性风险（高价卡小众难出手）

---

## 预测输出格式

```
📈 短期预测（6个月）：目标价 + 置信度 + 依据
📈 中期预测（1-2年）：目标价 + 置信度 + 依据
📈 长期预测（3-5年）：天花板价 + 置信度 + 依据

🆚 横向对比：与 X 球员的同类卡对比，当前价位是否合理

⚠️ 核心风险：Top 3 风险因素

💡 操作建议：买入/持有/减仓/卖出 + 目标卖出区间
```
