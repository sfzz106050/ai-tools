🔍 爬虫到底做了什么？

1. fetch(网页地址) → 拿到完整 HTML（就像你右键→查看网页源代码）
2. document.querySelectorAll('.item') → 找到所有商品卡片
3. card.querySelector('.name') → 从每个卡片里提取名字
4. card.querySelector('.price') → 从每个卡片里提取价格
5. 组装成干净的 JSON 数组 → 存进数据库

核心逻辑：爬虫不是魔法，就是"找到规律 → 按规律提取 → 结构化存储"。
真正爬卡淘时，步骤完全一样——只是网页结构更复杂、需要处理翻页和反爬。

你现在能用这个做什么：
- ❌ 不能爬卡淘（需要后端/服务器环境，绕过 CORS）
- ✅ 理解了底层逻辑 → 以后有 Node.js/Python 环境时，换个语言照样用
- ✅ 手动复制 HTML → 粘贴到工具里 → 自动提取（不用一行行看）
